### 🤖 명령어
- ``/기록 내용 [타이머]`` : 해당 내용에 대한 기록을 시작(기본 : 1시간)
  - ``타이머``는 h, m, s 단위로 시간 단위를 설정할 수 있음(단위 기본 : m)
- ``/정지`` : 진행 중인 기록을 정지 시킬 수 있음
- ``/중지`` : 진행 중인 기록을 중지 시킬 수 있음
- ``/재개`` : 중지된 기록을 진행 시킬 수 있음
